启动服务器（默认端口4000）

```hexo
$ hexo server
```

或

```hexo
$ hexo s
```

在指定端口（5000端口）上启动服务

```hexo
$ hexo s -p 5000
```

清除缓存文件 (db.json) 和已生成的静态文件 (public)

```hexo
$ hexo clean
```

生成静态文件

```hexo
$ hexo g
```
部署网站

```hexo
$ hexo d
```
