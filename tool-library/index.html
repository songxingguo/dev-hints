Rome，一个新的 JavaScript 工具库（英文）

Babel 创始人的新作品，把 JS 的主要工具都包括在一个库里。不需要 webpack、eslint、prettier、babel ，它都提供了。
它的核心思想是，把基于AST（抽象语法树）的所有功能都统一起来，不要每个工具自己做一次AST解析。
